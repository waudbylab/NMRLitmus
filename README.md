# NMR pH Calibration

**[waudbylab.github.io/nmr-pH](https://waudbylab.github.io/nmr-pH/)**

Estimates sample pH from the chemical shifts of NMR-active buffer resonances. Temperature, ionic strength, and chemical shift referencing errors can be refined alongside pH. Uncertainties are reported for all fitted parameters.

Multiple buffers and nuclei can be combined in a single fit. Observed shifts are assigned to predicted resonances automatically.
